# Emergency Landing Guidance System for Heavy-Lift Cargo UAVs

## Overview
This project presents a constraint-based emergency landing guidance system for a heavy-lift cargo UAV operating in BVLOS environments.

The system integrates:

- Multi-Criteria Decision Making (MCDM) landing-site selection
- Constraint-based trajectory generation
- Runtime safety monitoring
- Failure-aware adaptation
- Monte Carlo validation

## Key Results

- 200 Monte Carlo simulations completed
- Safety-envelope violations reduced from 200/200 to 0/200
- Peak impact energy reduced by 93.4%
- Mean peak energy reduced by 94.2%
